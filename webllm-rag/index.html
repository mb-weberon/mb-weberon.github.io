⚙️ Settings
Temperature

Higher = more creative, lower = more deterministic. Safe range: 0.1–0.9. Above 1.2 outputs become erratic. Default: 0.4

Repetition penalty

Penalises tokens already generated. Safe range: 1.0–1.4. Above 1.5 may cause incoherent output. Default: 1.3

Max tokens

Maximum response length. Safe range: 200–800 for speed. Higher values slow generation significantly. Default: 400

History exchanges

Prior Q&A pairs sent to LLM for follow-up context. Safe range: 1–3. Higher values grow prefill significantly. Default: 1

Enable RAG (Retrieval-Augmented Generation)
When off, queries go directly to the LLM with no retrieved context — useful for testing system prompts in isolation.
Passage char limit

Characters of each source chunk sent to the LLM. Higher = more context but slower prefill. Default: 200

Contact boost

Extra relevance score added to contact/footer chunks. Helps surface address and phone queries. Default: 0.15

Candidates multiplier

Fetch this many × sources before deduplicating to final count. Default: 3

Footnote snippet

Preview text shown under each source link in responses. Default: 120

Question Panel
Explore count

Questions shown in the side panel. Default: 20

Related count

Questions shown below each response. Default: 5
