📚 Site Notebook

⚙️ Settings
Temperature

Higher = more creative, lower = more deterministic. Default: 0.4

Repetition penalty

Penalises tokens already generated. Reduces looping. Default: 1.3

Max tokens

Maximum response length. Default: 400

History exchanges

Prior Q&A pairs sent to LLM for follow-up context. Higher = longer prefill. Default: 1

Passage char limit

Characters of each source chunk sent to the LLM. Higher = more context but slower prefill. Default: 200

Contact boost

Extra relevance score added to contact/footer chunks. Helps surface address and phone queries. Default: 0.15

Candidates multiplier

Fetch this many × sources before deduplicating to final count. Default: 3

Footnote snippet

Preview text shown under each source link in responses. Default: 120

⬇ Export

Download all current settings and models as a JSON file you can reload later.

📂 Import

Upload a previously exported settings file. A summary of changes will be shown before applying.

⚠️ Reset

Restore all settings and models to factory defaults. Chat history is kept. Page will reload.
